# Metrolist Kotlin Multiplatform Project

This project targets Android and Desktop (Windows, Linux, Mac) using Compose Multiplatform for UI.

## Modules
- `shared`: Shared business logic and UI code
- `androidApp`: Android-specific code and entry point
- `desktopApp`: Desktop-specific code and entry point

## Getting Started
1. Open the project in IntelliJ IDEA or Android Studio (KMP support recommended).
2. Sync Gradle and build the project.
3. Run the `desktopApp` or `androidApp` configuration as needed.

## Migration
Prepare to migrate existing Android code into `shared` and `androidApp` modules as appropriate.
